# Fake Product Review Detection

## 📌 Project Overview

Fake Product Review Detection is a Machine Learning based project that identifies whether a product review is genuine or fake.

With the growth of online shopping, fake reviews have become a major problem. This project uses Natural Language Processing (NLP) techniques and Machine Learning algorithms to analyze review text and classify it as a real or fake review.

---

## 🎯 Objective

The main objective of this project is:

- To detect fake product reviews automatically
- To reduce misleading information in online shopping platforms
- To build a text classification model using Machine Learning

---

## 🚀 Features

- Text-based review analysis
- Data preprocessing and cleaning
- Feature extraction using NLP
- Machine Learning model training
- Fake/Real review prediction
- Model saved for future predictions

---

## 🛠️ Technologies Used

- Python

- Natural Language Processing (NLP)

## 📄 File Description

### dataset.csv
Contains the product review dataset used for training and testing the model.

### model.py
Contains:
- Data loading
- Data preprocessing
- Model training
- Model evaluation
- Saving trained model

### model.pkl
Saved machine learning model used for prediction.

### vectorizer.pkl
Saved text vectorizer used to convert review text into numerical features.
